# prototype-DasDa

Alle Ansichten mobil testen.

#How to run

1. [NODEjs](https://nodejs.org/en/download/) installieren
2. im Ordner "npm install" ausführen
3. im Ordner "npm run dev" ausführen (cmd offen lassen)
4. http://localhost:8000/ aufrufen
